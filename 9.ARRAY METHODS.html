LETS SEE ABOUT ARRAY METHODS

1.push
add element at last and return new length

2.pop
delete element at last and return new length

3.shift
delete elements from front

4.unshift
add elements at front

5.delete
to delete elements but leaves undefined elements better to use splice

6.splice
to remove the desired element

7.splice insert
to insert the desired element, multiple values can also be inserted

8.slice
to get range of elements same as used in string

9.reverse
to reverse the array

10.join
to convert array to string

11.split
to convert string to array

12.concat
to join two array
